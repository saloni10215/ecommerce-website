# 🛒 ShopEasy - E-Commerce Website

A modern and responsive **E-Commerce Website** built using **HTML, CSS, and JavaScript**. This project demonstrates core front-end development concepts such as DOM manipulation, event handling, Local Storage, dynamic UI updates, and shopping cart functionality.

---

## 📌 Features

- 🛍️ Display products dynamically
- 🔍 Search products by name
- 📂 Filter products by category
- ↕️ Sort products
  - Price: Low to High
  - Price: High to Low
  - Name: A–Z
  - Name: Z–A
- 🛒 Add products to cart
- ❌ Remove products from cart
- ➕ Increase product quantity
- ➖ Decrease product quantity
- 🔢 Dynamic cart item counter
- 💰 Automatic total price calculation
- 💾 Save cart using Local Storage
- 🔔 Toast notification when adding products
- 📱 Responsive design for different screen sizes
